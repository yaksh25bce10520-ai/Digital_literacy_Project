📄 README – Digital Literacy Infographic
📌 Project Title

Digital Literacy for Students – Infographic

🛠️ Tool Used

This infographic was created using Canva, a free and user-friendly online design tool. Canva provides a variety of templates, icons, and layouts that make it easy to design visually appealing content without advanced design skills.

🎯 Description of the Design

The infographic presents important concepts of digital literacy in a simple and visually structured format. It is divided into four main sections: What is Digital Literacy, Useful Digital Tools, Safe Internet Practices, and Email Etiquette. Each section uses icons and short bullet points to make the information easy to understand. The design uses different colors for each section to improve clarity and visual appeal. It highlights essential skills such as using technology effectively, maintaining online safety, utilizing digital tools like Google Docs and Zoom, and following proper email communication practices.

💡 Learning Experience

While creating this infographic, one interesting aspect was selecting suitable icons and colors to match each section. It helped in making the design more engaging. However, one challenge I faced was organizing all the information in a limited space without making the infographic look crowded. Overall, this project improved my understanding of presenting information in a clear, creative, and visually effective way.
